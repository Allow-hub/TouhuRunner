本アプリ（以下「本ゲーム」）では、ユーザーの個人情報を収集することはありません。

1. 収集する情報

本ゲームはオフラインで動作し、個人を特定できる情報や通信による情報送信は行いません。

2. 第三者への情報提供

広告や外部解析ツールを使用していないため、情報を第三者に提供することはありません。
