# Playerの操作

## 機能要件

- ボタンを押したときに、プレイヤーを左右に1レーン動かす
- 4レーンを配列で管理する
- プレイヤーが前進する際、カメラも子オブジェクトとして追従させる
- プレイヤーはTransformを直接操作するのではなく、物理挙動で実装すること
- delta.timeを使用する場合はGameManager.Iを経由すること
→その方がポーズが簡単だから

- 名前空間はTechC.Main.Player等にすること


## 操作方法

- InputSystemをボタン操作で制御する

## 必要なスクリプト一覧

| スクリプト名           | 役割・説明                                      |
|------------------------|------------------------------------------------|
| PlayerController.cs    | プレイヤーのレーン移動・前進・物理挙動の制御    |
| LaneController.cs         | レーン情報の管理（配列でレーン位置を保持）      |
| CameraFollow.cs        | カメラがプレイヤーに追従する処理                |
| InputHandler.cs        | InputSystemを使ったボタン入力の管理             |
